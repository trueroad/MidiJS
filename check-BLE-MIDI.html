Bluetooth MIDI (BLE-MIDI)確認ページ

注意
iPhoneやiPadをお使いの場合、 通常のSafariやChromeではBluetooth MIDIが使えません。
Bluefy - Web BLE Browser をインストールして、それで開いてください。

Android をお使いの方は通常の Chrome で開いてください。

使い方

  1. ピアノの電源を入れてください
  2. ボタンを タップしてください
  3. 「ペア設定を要求しています」や「ペア設定を求めています」 などの画面が出てきます
    • この画面が出てこない場合は以下をご確認ください
      • お使いのスマートフォン・タブレット端末のBluetoothがONになっていることを確認してください
      • アプリをアップデートしてもう一度最初から試してみてください
      • iPhone/iPadをお使いの場合はSafariやChromeではなくて Bluefy - Web BLE Browser を使っているか確認してください
    • お使いのスマートフォン・タブレット端末が古い場合には使えないことがあります
  4. しばらく待ってみて画面に選択肢が出てきたら、それをタップしてください
    • しばらく待っても「対応デバイスが見つかりません」や「互換性のあるデバイスが見つかりません」などの表示のまま選択肢が出ない場合は以下をご確認ください
      • 電子ピアノの電源が入っているか
      • 電子ピアノの近くにいるか
    • それでも出てこない場合には、お使いの電子ピアノがBluetooth MIDI (BLE-MIDI)非対応だと思われます
  5. 「ペア設定」や「ペアリング」などのボタンが有効になるのでタップしてください
  6. 画面の下の方の結果欄に「Bluetooth MIDI (BLE-MIDI)対応です」という表示が出てくれば完了です
